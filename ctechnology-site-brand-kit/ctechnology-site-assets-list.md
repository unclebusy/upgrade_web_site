# Список ассетов для сайта Ctechnology

## Логотипы
1. `logo-main-horizontal`
   - основной логотип для header и footer
2. `logo-mark`
   - знак без текста для favicon, preloader, соцсетей
3. `logo-white`
   - белая версия для темного фона
4. `logo-black`
   - монохромная темная версия для светлого фона
5. `logo-icon-square`
   - квадратная версия для app icon / Open Graph fallback
